## Features

**Dashboard Screen**
- Quick overview of recent transactions (last 5 entries)
- Visual expense breakdown with interactive chart
- Fast access to transaction details

**Transactions Screen**
- Complete transaction history
- Organized by month with collapsible sections
- Easy navigation through historical data
- Detailed view of all recorded expenses

**Theme Support**
- Full light mode and dark mode support
- Seamless theme switching
- Consistent design across all screens
